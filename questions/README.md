### Topics covered so far
- **Security:** private S3 via VPC endpoint + encryption enforcement · cross-account KMS · CloudFront OAC · Secrets Manager rotation · NACL vs security group (deny) · AWS WAF
- **Resilient:** RDS Multi-AZ failover · SQS decoupling · stateless sessions · DR strategy (pilot light) · Aurora Global Database
- **High-Performing:** DynamoDB DAX · Global Accelerator vs CloudFront · RDS read replicas · EBS io2 · S3 Transfer Acceleration
- **Cost-Optimized:** S3 Intelligent-Tiering · Savings Plans · Spot · S3 gateway endpoint (NAT cost)

### How to self-test
Answers live inside `▶ Reveal` blocks — they stay collapsed until you click, so you can attempt each question first.

### Viewing the diagrams
Diagrams use [Mermaid](https://mermaid.js.org/); GitHub renders them automatically. Elsewhere, use a Mermaid-aware viewer or paste a block into [mermaid.live](https://mermaid.live).
